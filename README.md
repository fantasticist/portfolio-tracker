# Portfolio Tracker
This is a portfolio tracker app built using [Next.js](https://nextjs.org/) + [Turbopack](https://turbo.build/pack) + [Tailwindcss](https://tailwindcss.com).

## Installation
```shell
pnpm install
```

## Development
```shell
pnpm dev
```

## Build
```shell
pnpm build
```
